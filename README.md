# 二维激光雷达占据栅格地图建图仿真项目

## 项目简介
本项目实现了一个二维仿真环境下的占据栅格地图构建系统。它模拟了机器人携带激光雷达在环境中运动，并根据传感器观测数据实时构建地图。项目包含了环境构建、机器人轨迹预设、激光雷达模拟、坐标变换、占据栅格地图更新以及动态可视化等完整流程。


## 目录结构
```
robot_mapping/

├── config.py                # 全局参数配置
├── environment.py           # 二维仿真环境构建
├── robot.py                 # 机器人位姿管理及预设轨迹
├── lidar_sensor.py          # 模拟二维激光雷达
├── occupancy_map.py         # 占据栅格地图
├── main.py                  # 主程序：轨迹执行、建图、动态显示、结果保存

│   ├── environment.png
│   ├── trajectory.png
│   ├── lidar_scan.png
│   ├── process_frame_*.png
│   ├── final_map.png
│   └── analysis_table.txt
└── README.md                # 本文件
```


## 环境要求
- Python 3.7+
- 依赖库：
  ```bash
  numpy
  matplotlib
  ```

## 主要模块说明

### 1. `config.py`
配置所有全局参数，包括：
- 地图大小
- 栅格分辨率
- 激光雷达角度范围、角度步长、最大测距、噪声标准差
- 机器人起始位姿

### 2. `environment.py`
构建一个 10m×10m 的二维环境，包含：
- 边界墙体
- 三个组合型障碍物（L形、T形、竖条）

提供 `world_to_grid`、`grid_to_world` 坐标转换方法和可视化接口。

### 3. `robot.py`
- 管理机器人位置 `(x, y, θ)` 和轨迹记录
- 提供预设的固定往复式轨迹（`generate_coverage_trajectory`）覆盖自由空间
- 支持轨迹绘制

### 4. `lidar_sensor.py`
模拟二维激光雷达：
- 射线投射算法，步进检测障碍物
- 支持角度范围、角度步长、最大测距、测距噪声（高斯）配置
- 返回每条光束的测距值

### 5. `occupancy_map.py`
占据栅格地图核心类：
- 使用对数几率（Log‑Odds）进行二值贝叶斯更新
- 每次扫描后更新障碍物端点和射线路径上的空闲栅格
- 提供获取概率图（`get_prob_map`）和二值图（`get_map`，含未知标记 -1）的方法

### 6. `main.py`
主程序执行流程：
1. 生成仿真环境并保存 `environment.png`
2. 生成机器人轨迹并保存 `trajectory.png`
3. 初始化激光雷达和占据栅格地图
4. 对每个轨迹点执行扫描，更新地图，并动态显示建图过程（机器人红点 + 轨迹线）
5. 保存过程图（每50帧）
6. 保存最终地图（`final_map.png`，黑=空闲，白=障碍，灰=未知）
7. 计算地图正确率和未知区域比例
8. 生成数据分析表（`analysis_table.txt`）
9. 选取一个中间位姿，保存传感器扫描效果图（`lidar_scan.png`）


## 如何运行

1. 确保所有 `.py` 文件位于同一目录，并安装依赖库。
2. 在项目根目录下运行：
   ```bash
   python main.py
   ```
3. 程序将弹出动态窗口，显示地图构建过程。
4. 运行结束后，所有输出文件自动保存至 `results/` 文件夹。


## 输出结果说明

| 文件 | 描述 |
|------|------|
| `environment.png` | 仿真环境真实地图（墙体+障碍物） |
| `trajectory.png` | 预设机器人运动轨迹（叠加在环境图上） |
| `lidar_scan.png` | 在某个位姿下的激光扫描效果（绿色实线为探测到障碍，红色虚线为未探测到障碍） |
| `process_frame_*.png` | 地图构建过程中的关键帧 |
| `final_map.png` | 最终占据栅格地图 |
| `analysis_table.txt` | 实验结果数据记录（分辨率、角度步长、测距、扫描次数、正确率、未知比例、耗时） |


## 参数调整建议

您可以通过修改 `config.py` 中的参数进行实验对比：

- **栅格分辨率** (`RESOLUTION`)：影响地图精细度和计算时间。
- **激光角度步长** (`LIDAR_ANGLE_STEP`)：控制扫描密度，步长越小轮廓越清晰。
- **最大测距** (`LIDAR_RANGE`)：影响环境覆盖范围和未知区域比例。
- **测距噪声** (`LIDAR_NOISE`)：模拟传感器噪声对建图质量的影响。

每次修改参数后重新运行 `main.py`，结果将追加到 `analysis_table.txt` 中，方便对比。
